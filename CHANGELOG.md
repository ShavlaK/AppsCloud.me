## [Unreleased]

### Добавлено
- Веб-интерфейс (Dashboard) с 4 вкладками: Статус, Конфигурация, Логи, Отчёты
- REST API для управления всеми функциями
- SQLite хранилище с историей цен (30 дней)
- Генератор ежемесячных отчётов (рост/падение/новые/исчезнувшие позиции)
- Система логирования с ротацией файлов
- User API модуль (Telethon) для навигации по ботам через кнопки — ЗАКОММЕНТИРОВАН
- Демо-режим для тестирования без токенов (`python demo.py`)
- Тестовая среда (pytest + моки)
- GitHub Actions workflow для авто-сборки
- Скрипты сборки: .exe (Windows), .dmg (macOS), .deb (Linux)
- `pyproject.toml` для `pip install`
- CONTRIBUTING.md, LICENSE
